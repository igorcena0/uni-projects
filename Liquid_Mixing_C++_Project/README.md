PL: Projekt polegał na stworzeniu symulacji mieszania różnych substancji w kubku i wyświetlaniu podstawowych informacji na ten temat. Dodane zostały funkcje umożliwiające np. określanie udziału procentowego substancji w kubku, na podstawie którego dolewana była brakująca ilość substancji. Pliki zostały podzielone na źródłowe i nagłówkowe klas substancji i kubka oraz plik main - "kubki".<br><br>
EN: The project involved creating a simulation of mixing different substances in a cup and displaying basic information about it. Functions were added to allow, for example, specifying the percentage share of a substance in the cup, based on which the missing amount of the substance was added. The files were divided into source and header files for the substance and cup classes, as well as the main file - cups.
